# Графо-моторные навыки - Система упражнений для детей

Интерактивное веб-приложение для развития графо-моторных навыков у детей.

## 📖 Описание

Это система упражнений, разработанная для помощи детям в развитии навыков письма и мелкой моторики через интерактивные задания.

## 🚀 Возможности

- **Начать занятие** - автоматический режим прохождения упражнений
- **Выбрать упражнение** - ручной выбор конкретного задания
- **Мои результаты** - отслеживание прогресса и достижений
- **Для взрослого** - информация для родителей и педагогов
- **Выйти и очистить данные** - сброс прогресса

## 📋 Установка и запуск

1. Скачайте или клонируйте репозиторий
2. Откройте `index.html` в любом современном браузере

```bash
# Или используйте локальный сервер
python -m http.server 8000
```

Затем откройте в браузере: `http://localhost:8000`
